# Codex Mate
Codex Mate 显示下方信息 <br>
- 5小时余量
- 7天余量
- 统计每天、每周、每月、总计的tokens使用量 <br>
通过iCloudkit同步到iOS端，支持iOS app 和 widget 显示余量。 <br>
