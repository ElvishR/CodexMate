# Codex Mate
Codex Mate 显示下方信息 <br>
- 5小时余量
- 7天余量
- 统计每天、每周、每月、总计的tokens使用量 <br>
通过iCloudkit同步到iOS端，支持iOS app 、apple watch 和 widget 显示余量。 <br>
所有数据仅保存在本地及你的私人 icloud 里，开发者不获取任何数据。 <br>
